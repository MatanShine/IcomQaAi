Prompt Management Page Layout

Single page with three sections: prompt selector, version list, and comparison dashboard

A

Top-Down Layout

Prompt type selector at top, then a split view: left side shows version list + editor, right side shows comparison metrics

B

Tab-Based Layout

Tabs for each prompt type at the top. Below: three sub-tabs — "Versions" (list + editor), "Test" (set testing version + run in Test Agent), "Compare" (metrics dashboard)

C

List-Detail Layout

Left sidebar lists prompt types + versions (tree view). Selecting a version opens it in a main content area with editor, status actions, and inline metrics below

Wireframe: Option B (Recommended)

Tab-based layout keeps concerns separated while staying on one page
